# RobloxUDMUXCrasher

RobloxUDMUXCrasher is a tool designed for educational and research purposes to demonstrate vulnerabilities in Roblox multiplayer server stability. Use responsibly and **only in environments you own or have explicit permission to test**.

## Features

- Simulates methods that can crash or disrupt Roblox servers.
- Demonstrates potential flaws in remote event handling, denial of service scenarios, or stress tests.
- Allows user configuration of payload types and target endpoints.

## Disclaimer

This project is intended for educational and ethical penetration testing purposes only:
- **Do not use this tool against servers without explicit authorization.**
- The author is not responsible for misuse or any damage resulting from the use of this software.
- Always follow Roblox's Terms of Service and ethical hacking guidelines.

## Getting Started

### Prerequisites

- Roblox Studio installed.
- Basic knowledge of Lua scripting.
- Access to a Roblox test environment or a private server.

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/y0nyy/RobloxUDMUXCrasher.git
    ```
2. Have NodeJS installed

### Usage

1. Review all scripts and understand each action before running.
2. Run scripts only in safe, authorized environments.
3. Modify .env file with your Bot token, Roblox Account Cookie & Owner IDS
4. Have a VPS with SSH Credentials
5. Insert them into attackuser.js config
